# M1.2 LiteStore Implementation Plan

> **Status:** Shipped (v0.1.2)

**Goal:** Add zero-Docker storage and inference backends (LiteStore + OllamaEngine) so Corvia runs with just `cargo install` and a local Ollama instance.

**Architecture:** LiteStore implements `QueryableStore` using hnsw_rs for HNSW vector search and Redb for metadata indexing, with JSON files in `.corvia/knowledge/` as the git-tracked source of truth. OllamaEngine implements `InferenceEngine` using Ollama's OpenAI-compatible embedding API. Config gains `store_type` (lite/surrealdb) and `provider` (ollama/vllm) enum fields. Factory functions dispatch the correct backend based on config.

**Tech Stack:** hnsw_rs 0.3 (vector search), redb 2 (metadata KV store), reqwest (HTTP for Ollama), serde_json (knowledge files), tempfile (test fixtures)

**Key design decisions:** D35 (two-tier storage), D36 (hnsw_rs), D40 (Ollama default), D41 (git-tracked JSON), D42 (M1.2 scope)

**What M1.2 does NOT include** (deferred per D42): petgraph graph overlay (M3), bi-temporal Redb overlay (M3), RBAC isolation (M2), `corvia upgrade` migration (M3), knowledge file writing for SurrealStore (future).

```rust
use async_trait::async_trait;
use corvia_common::errors::{CorviaError, Result};
use serde::{Deserialize, Serialize};

pub struct OllamaEngine {
    url: String,
    model: String,
    dimensions: usize,
    client: reqwest::Client,
}

#[derive(Serialize)]
struct EmbedRequest {
    model: String,
    input: Vec<String>,
}

#[derive(Deserialize)]
struct EmbedResponse {
    data: Vec<EmbedData>,
}

#[derive(Deserialize)]
struct EmbedData {
    embedding: Vec<f32>,
    index: usize,
}

impl OllamaEngine {
    pub fn new(url: &str, model: &str, dimensions: usize) -> Self {
        Self {
            url: url.to_string(),
            model: model.to_string(),
            dimensions,
            client: reqwest::Client::new(),
        }
    }
}

#[async_trait]
impl super::traits::InferenceEngine for OllamaEngine {
    async fn embed(&self, text: &str) -> Result<Vec<f32>> {
        let results = self.embed_batch(&[text.to_string()]).await?;
        results.into_iter().next()
            .ok_or_else(|| CorviaError::Embedding("Empty embedding response from Ollama".into()))
    }

    async fn embed_batch(&self, texts: &[String]) -> Result<Vec<Vec<f32>>> {
        let url = format!("{}/v1/embeddings", self.url);
        let request = EmbedRequest {
            model: self.model.clone(),
            input: texts.to_vec(),
        };

        let response = self.client.post(&url)
            .json(&request)
            .send()
            .await
            .map_err(|e| CorviaError::Embedding(format!(
                "Ollama request failed (is Ollama running at {}?): {e}", self.url
            )))?;

        if !response.status().is_success() {
            let status = response.status();
            let body = response.text().await.unwrap_or_default();
            return Err(CorviaError::Embedding(format!("Ollama returned {status}: {body}")));
        }

        let embed_response: EmbedResponse = response.json().await
            .map_err(|e| CorviaError::Embedding(format!("Failed to parse Ollama response: {e}")))?;

        let mut data = embed_response.data;
        data.sort_by_key(|d| d.index);

        Ok(data.into_iter().map(|d| d.embedding).collect())
    }

    fn dimensions(&self) -> usize {
        self.dimensions
    }
}
```

```rust
use corvia_common::errors::{CorviaError, Result};
use corvia_common::types::KnowledgeEntry;
use std::path::{Path, PathBuf};
use uuid::Uuid;

/// Path: {data_dir}/knowledge/{scope_id}/{entry_id}.json
fn entry_path(data_dir: &Path, scope_id: &str, entry_id: &Uuid) -> PathBuf {
    data_dir.join("knowledge").join(scope_id).join(format!("{}.json", entry_id))
}

fn scope_dir(data_dir: &Path, scope_id: &str) -> PathBuf {
    data_dir.join("knowledge").join(scope_id)
}

/// Write a KnowledgeEntry as a JSON file.
pub fn write_entry(data_dir: &Path, entry: &KnowledgeEntry) -> Result<()> {
    let path = entry_path(data_dir, &entry.scope_id, &entry.id);
    if let Some(parent) = path.parent() {
        std::fs::create_dir_all(parent)
            .map_err(|e| CorviaError::Storage(format!("Failed to create dir: {e}")))?;
    }
    let json = serde_json::to_string_pretty(entry)
        .map_err(|e| CorviaError::Storage(format!("Failed to serialize entry: {e}")))?;
    std::fs::write(&path, json)
        .map_err(|e| CorviaError::Storage(format!("Failed to write {}: {e}", path.display())))?;
    Ok(())
}

/// Read a single KnowledgeEntry from its JSON file.
pub fn read_entry(data_dir: &Path, scope_id: &str, entry_id: &Uuid) -> Result<KnowledgeEntry> {
    let path = entry_path(data_dir, scope_id, entry_id);
    let json = std::fs::read_to_string(&path)
        .map_err(|e| CorviaError::NotFound(format!("{}: {e}", path.display())))?;
    serde_json::from_str(&json)
        .map_err(|e| CorviaError::Storage(format!("Failed to parse {}: {e}", path.display())))
}

/// Read all KnowledgeEntries in a scope directory.
pub fn read_scope(data_dir: &Path, scope_id: &str) -> Result<Vec<KnowledgeEntry>> {
    let dir = scope_dir(data_dir, scope_id);
    if !dir.exists() {
        return Ok(Vec::new());
    }
    let mut entries = Vec::new();
    for file in std::fs::read_dir(&dir)
        .map_err(|e| CorviaError::Storage(format!("Failed to read dir: {e}")))?
    {
        let file = file.map_err(|e| CorviaError::Storage(e.to_string()))?;
        let path = file.path();
        if path.extension().and_then(|e| e.to_str()) == Some("json") {
            let json = std::fs::read_to_string(&path)
                .map_err(|e| CorviaError::Storage(format!("Failed to read {}: {e}", path.display())))?;
            if let Ok(entry) = serde_json::from_str::<KnowledgeEntry>(&json) {
                entries.push(entry);
            }
        }
    }
    Ok(entries)
}

/// Read all KnowledgeEntries across all scopes.
pub fn read_all(data_dir: &Path) -> Result<Vec<KnowledgeEntry>> {
    let knowledge_dir = data_dir.join("knowledge");
    if !knowledge_dir.exists() {
        return Ok(Vec::new());
    }
    let mut entries = Vec::new();
    for scope_entry in std::fs::read_dir(&knowledge_dir)
```

This is the largest task. LiteStore combines three storage mechanisms:

**Note on hnsw_rs API:** The exact type signatures for `Hnsw`, `insert`, `search`, and `Neighbour` fields are based on docs research. If compile errors occur:
- `Hnsw::new()` takes `(max_nb_connection, max_elements, nb_layer, ef_construction, DistCosine{})`
- `hnsw.insert()` takes a tuple of `(&Vec<f32>, usize)` — the data and an ID
- `hnsw.search()` returns `Vec<Neighbour>` — check field names: likely `d_id` (usize) and `distance` (f32)

```rust
impl LiteStore {
    pub fn open(data_dir: &Path, dimensions: usize) -> Result<Self> {
        std::fs::create_dir_all(data_dir)
            .map_err(|e| CorviaError::Storage(format!("Failed to create data dir: {e}")))?;

        let db_path = data_dir.join("metadata.redb");
        let db = Database::create(&db_path)
            .map_err(|e| CorviaError::Storage(format!("Failed to open Redb: {e}")))?;

        // Load next_hnsw_id from Redb (or start at 0)
        let next_id = {
            let read_txn = db.begin_read()
                .map_err(|e| CorviaError::Storage(format!("Redb read error: {e}")))?;
            if let Ok(table) = read_txn.open_table(META) {
                table.get("next_hnsw_id").ok().flatten()
                    .map(|v| v.value())
                    .unwrap_or(0)
            } else {
                0
            }
        };

        let max_elements = 100_000;
        let max_nb_connection = 24;
        let nb_layer = 16;
        let ef_construction = 400;

        let hnsw = Hnsw::<f32, DistCosine>::new(
            max_nb_connection,
            max_elements,
            nb_layer,
            ef_construction,
            DistCosine {},
        );

        let store = Self {
            data_dir: data_dir.to_path_buf(),
            db,
            hnsw: Arc::new(Mutex::new(hnsw)),
            next_hnsw_id: AtomicU64::new(next_id),
            dimensions,
        };

        info!("LiteStore opened at {} (dim={dimensions})", data_dir.display());
        Ok(store)
    }

    fn allocate_hnsw_id(&self) -> u64 {
        self.next_hnsw_id.fetch_add(1, Ordering::SeqCst)
    }

    fn persist_next_id(&self) -> Result<()> {
        let write_txn = self.db.begin_write()
            .map_err(|e| CorviaError::Storage(format!("Redb write error: {e}")))?;
        {
            let mut table = write_txn.open_table(META)
                .map_err(|e| CorviaError::Storage(e.to_string()))?;
            table.insert("next_hnsw_id", self.next_hnsw_id.load(Ordering::SeqCst))
                .map_err(|e| CorviaError::Storage(e.to_string()))?;
        }
        write_txn.commit()
            .map_err(|e| CorviaError::Storage(format!("Redb commit error: {e}")))?;
        Ok(())
    }

    /// Flush HNSW index to disk for persistence.
    pub fn flush_hnsw(&self) -> Result<()> {
        let hnsw = self.hnsw.lock().unwrap();
        hnsw.file_dump(&self.data_dir, "index")
            .map_err(|e| CorviaError::Storage(format!("Failed to dump HNSW index: {e}")))?;
        info!("HNSW index flushed to disk");
        Ok(())
    }

    /// Internal: index an entry into HNSW + Redb (without writing JSON file).
    fn index_entry(&self, entry: &KnowledgeEntry, embedding: &[f32]) -> Result<()> {
        let hnsw_id = self.allocate_hnsw_id();
        let uuid_str = entry.id.to_string();

        // Insert into HNSW
        {
            let hnsw = self.hnsw.lock().unwrap();
            let data = embedding.to_vec();
            hnsw.insert((&data, hnsw_id as usize));
        }

        // Insert into Redb
        let entry_json = serde_json::to_vec(entry)
            .map_err(|e| CorviaError::Storage(format!("Serialize error: {e}")))?;

        let write_txn = self.db.begin_write()
            .map_err(|e| CorviaError::Storage(format!("Redb write error: {e}")))?;
        {
            let mut entries_table = write_txn.open_table(ENTRIES)
                .map_err(|e| CorviaError::Storage(e.to_string()))?;
            entries_table.insert(uuid_str.as_str(), entry_json.as_slice())
                .map_err(|e| CorviaError::Storage(e.to_string()))?;

            let mut scope_table = write_txn.open_table(SCOPE_INDEX)
                .map_err(|e| CorviaError::Storage(e.to_string()))?;
            let scope_key = format!("{}:{}", entry.scope_id, uuid_str);
            scope_table.insert(scope_key.as_str(), uuid_str.as_str())
                .map_err(|e| CorviaError::Storage(e.to_string()))?;

            let mut h2u = write_txn.open_table(HNSW_TO_UUID)
                .map_err(|e| CorviaError::Storage(e.to_string()))?;
            h2u.insert(hnsw_id, uuid_str.as_str())
                .map_err(|e| CorviaError::Storage(e.to_string()))?;

            let mut u2h = write_txn.open_table(UUID_TO_HNSW)
                .map_err(|e| CorviaError::Storage(e.to_string()))?;
            u2h.insert(uuid_str.as_str(), hnsw_id)
                .map_err(|e| CorviaError::Storage(e.to_string()))?;
        }
        write_txn.commit()
            .map_err(|e| CorviaError::Storage(format!("Redb commit error: {e}")))?;

        Ok(())
    }

    /// Rebuild indexes from knowledge JSON files. Clears Redb and re-indexes all entries.
    pub fn rebuild_from_files(&self) -> Result<usize> {
        let entries = crate::knowledge_files::read_all(&self.data_dir)?;
        if entries.is_empty() {
            info!("No knowledge files found, nothing to rebuild");
            return Ok(0);
        }

        // Reset HNSW
        self.next_hnsw_id.store(0, Ordering::SeqCst);
        let max_elements = entries.len().max(100_000);
        let new_hnsw = Hnsw::<f32, DistCosine>::new(24, max_elements, 16, 400, DistCosine {});
        {
            let mut hnsw = self.hnsw.lock().unwrap();
            *hnsw = new_hnsw;
        }

        // Re-index each entry (Redb tables will be overwritten)
        let mut count = 0;
        for entry in &entries {
            if let Some(embedding) = &entry.embedding {
                self.index_entry(entry, embedding)?;
                count += 1;
            }
        }

        self.persist_next_id()?;
        self.flush_hnsw()?;
        info!("Rebuilt LiteStore from {count} knowledge files");
        Ok(count)
    }
}
```

### Step 5: Implement QueryableStore trait

Add the trait implementation:

```rust
#[async_trait]
impl crate::traits::QueryableStore for LiteStore {
    async fn init_schema(&self) -> Result<()> {
        // Create all Redb tables (they're created on first open in a write txn)
        let write_txn = self.db.begin_write()
            .map_err(|e| CorviaError::Storage(format!("Redb write error: {e}")))?;
        {
            write_txn.open_table(ENTRIES).map_err(|e| CorviaError::Storage(e.to_string()))?;
            write_txn.open_table(SCOPE_INDEX).map_err(|e| CorviaError::Storage(e.to_string()))?;
            write_txn.open_table(HNSW_TO_UUID).map_err(|e| CorviaError::Storage(e.to_string()))?;
            write_txn.open_table(UUID_TO_HNSW).map_err(|e| CorviaError::Storage(e.to_string()))?;
            write_txn.open_table(META).map_err(|e| CorviaError::Storage(e.to_string()))?;
        }
        write_txn.commit()
            .map_err(|e| CorviaError::Storage(format!("Redb commit error: {e}")))?;
        info!("LiteStore schema initialized");
        Ok(())
    }

    async fn insert(&self, entry: &KnowledgeEntry) -> Result<()> {
        let embedding = entry.embedding.as_ref()
            .ok_or_else(|| CorviaError::Storage("Entry must have embedding for insert".into()))?;

        // Write knowledge JSON file (git-tracked source of truth)
        crate::knowledge_files::write_entry(&self.data_dir, entry)?;

        // Index into HNSW + Redb
        self.index_entry(entry, embedding)?;

        // Persist HNSW ID counter
        self.persist_next_id()?;

        Ok(())
    }

    async fn search(
        &self,
        embedding: &[f32],
        scope_id: &str,
        limit: usize,
    ) -> Result<Vec<SearchResult>> {
        // D36: Post-filter after KNN retrieval.
        // Retrieve K×2 candidates, filter by scope, return top K.
        let fetch_count = (limit * 2).max(10);
        let ef_search = 64.max(fetch_count);

        let neighbours = {
            let hnsw = self.hnsw.lock().unwrap();
            hnsw.search(&embedding.to_vec(), fetch_count, ef_search)
        };

        let read_txn = self.db.begin_read()
            .map_err(|e| CorviaError::Storage(format!("Redb read error: {e}")))?;
        let h2u_table = read_txn.open_table(HNSW_TO_UUID)
            .map_err(|e| CorviaError::Storage(e.to_string()))?;
        let entries_table = read_txn.open_table(ENTRIES)
            .map_err(|e| CorviaError::Storage(e.to_string()))?;

        let mut results = Vec::new();
        for neighbour in &neighbours {
            // NOTE: Verify these field names against hnsw_rs docs.
            // The Neighbour struct likely has d_id (usize) and distance (f32).
            let hnsw_id = neighbour.d_id as u64;
            let distance = neighbour.distance;

            // Look up UUID from HNSW ID
            let uuid_str = match h2u_table.get(hnsw_id) {
                Ok(Some(v)) => v.value().to_string(),
                _ => continue,
            };

            // Look up entry from UUID
            let entry_bytes = match entries_table.get(uuid_str.as_str()) {
                Ok(Some(v)) => v.value().to_vec(),
                _ => continue,
            };

            let entry: KnowledgeEntry = match serde_json::from_slice(&entry_bytes) {
                Ok(e) => e,
                Err(_) => continue,
            };

            // Post-filter by scope
            if entry.scope_id != scope_id {
                continue;
            }

            let score = 1.0 - distance; // cosine distance → similarity
            results.push(SearchResult { entry, score });

            if results.len() >= limit {
                break;
            }
        }

        Ok(results)
    }

    async fn get(&self, id: &uuid::Uuid) -> Result<Option<KnowledgeEntry>> {
        let uuid_str = id.to_string();
        let read_txn = self.db.begin_read()
            .map_err(|e| CorviaError::Storage(format!("Redb read error: {e}")))?;
        let table = read_txn.open_table(ENTRIES)
            .map_err(|e| CorviaError::Storage(e.to_string()))?;

        match table.get(uuid_str.as_str()) {
            Ok(Some(v)) => {
                let entry: KnowledgeEntry = serde_json::from_slice(v.value())
                    .map_err(|e| CorviaError::Storage(format!("Deserialize error: {e}")))?;
                Ok(Some(entry))
            }
            Ok(None) => Ok(None),
            Err(e) => Err(CorviaError::Storage(format!("Redb get error: {e}"))),
        }
    }

    async fn count(&self, scope_id: &str) -> Result<u64> {
        let prefix = format!("{scope_id}:");
        let prefix_end = format!("{scope_id};"); // ';' is next ASCII char after ':'

        let read_txn = self.db.begin_read()
            .map_err(|e| CorviaError::Storage(format!("Redb read error: {e}")))?;
        let table = read_txn.open_table(SCOPE_INDEX)
            .map_err(|e| CorviaError::Storage(e.to_string()))?;

        let mut count = 0u64;
        let range = table.range(prefix.as_str()..prefix_end.as_str())
            .map_err(|e| CorviaError::Storage(e.to_string()))?;
        for _ in range {
            count += 1;
        }

        Ok(count)
    }

    async fn delete_scope(&self, scope_id: &str) -> Result<()> {
        let prefix = format!("{scope_id}:");
        let prefix_end = format!("{scope_id};");

        // Collect UUIDs to delete
        let uuids_to_delete: Vec<String> = {
            let read_txn = self.db.begin_read()
                .map_err(|e| CorviaError::Storage(e.to_string()))?;
            let table = read_txn.open_table(SCOPE_INDEX)
                .map_err(|e| CorviaError::Storage(e.to_string()))?;
            table.range(prefix.as_str()..prefix_end.as_str())
                .map_err(|e| CorviaError::Storage(e.to_string()))?
                .filter_map(|r| r.ok().map(|(_, v)| v.value().to_string()))
                .collect()
        };

        if uuids_to_delete.is_empty() {
            return Ok(());
        }

        // Delete from Redb tables
        let write_txn = self.db.begin_write()
            .map_err(|e| CorviaError::Storage(e.to_string()))?;
        {
            let mut entries_table = write_txn.open_table(ENTRIES)
                .map_err(|e| CorviaError::Storage(e.to_string()))?;
            let mut scope_table = write_txn.open_table(SCOPE_INDEX)
                .map_err(|e| CorviaError::Storage(e.to_string()))?;
            let mut u2h_table = write_txn.open_table(UUID_TO_HNSW)
                .map_err(|e| CorviaError::Storage(e.to_string()))?;
            let mut h2u_table = write_txn.open_table(HNSW_TO_UUID)
                .map_err(|e| CorviaError::Storage(e.to_string()))?;

            for uuid_str in &uuids_to_delete {
                entries_table.remove(uuid_str.as_str()).ok();
                let scope_key = format!("{scope_id}:{uuid_str}");
                scope_table.remove(scope_key.as_str()).ok();
                if let Ok(Some(hnsw_id_guard)) = u2h_table.remove(uuid_str.as_str()) {
                    h2u_table.remove(hnsw_id_guard.value()).ok();
                }
            }
        }
        write_txn.commit()
            .map_err(|e| CorviaError::Storage(e.to_string()))?;

        // Delete knowledge files for this scope
        crate::knowledge_files::delete_scope_files(&self.data_dir, scope_id)?;

        info!("Deleted {} entries for scope '{scope_id}'", uuids_to_delete.len());
        Ok(())
    }
}
```

```rust
/// Create the appropriate InferenceEngine based on config.
pub fn create_engine(config: &CorviaConfig) -> Box<dyn traits::InferenceEngine> {
    match config.embedding.provider {
        InferenceProvider::Ollama => Box::new(ollama_engine::OllamaEngine::new(
            &config.embedding.url,
            &config.embedding.model,
            config.embedding.dimensions,
        )),
        InferenceProvider::Vllm => Box::new(embedding_pipeline::VllmEngine::new(
            &config.embedding.url,
            &config.embedding.model,
            config.embedding.dimensions,
        )),
    }
}

/// Create the appropriate QueryableStore based on config.
pub async fn create_store(config: &CorviaConfig) -> Result<Box<dyn traits::QueryableStore>> {
    match config.storage.store_type {
        StoreType::Lite => {
            let store = lite_store::LiteStore::open(
                std::path::Path::new(&config.storage.data_dir),
                config.embedding.dimensions,
            )?;
            Ok(Box::new(store))
        }
        StoreType::Surrealdb => {
            let store = knowledge_store::SurrealStore::connect(
                config.storage.surrealdb_url.as_deref().unwrap_or("127.0.0.1:8000"),
                config.storage.surrealdb_ns.as_deref().unwrap_or("corvia"),
                config.storage.surrealdb_db.as_deref().unwrap_or("main"),
                config.storage.surrealdb_user.as_deref().unwrap_or("root"),
                config.storage.surrealdb_pass.as_deref().unwrap_or("root"),
                config.embedding.dimensions,
            ).await?;
            Ok(Box::new(store))
        }
    }
}
```

```rust
async fn cmd_ingest(path: &str) -> Result<()> {
    let config = load_config()?;
    let store = connect_store(&config).await?;
    let engine = connect_engine(&config);

    let adapter = GitAdapter::new();
    println!("Ingesting {}...", path);

    let entries = adapter.ingest(path).await?;
    let total = entries.len();
    println!("Parsed {} chunks. Embedding and storing...", total);

    let mut stored = 0;
    for batch in entries.chunks(corvia_kernel::introspect::EMBED_BATCH_SIZE) {
        let texts: Vec<String> = batch.iter().map(|e| e.content.clone()).collect();
        let embeddings = engine.embed_batch(&texts).await?;

        for (entry, embedding) in batch.iter().zip(embeddings) {
            let mut entry = entry.clone();
            entry.embedding = Some(embedding);
            store.insert(&entry).await?;
            stored += 1;
        }
        println!("  {stored}/{total} chunks stored");
    }

    println!("Done. {stored} chunks ingested from {path}.");
    println!("Next: corvia search \"your query\"");
    Ok(())
}
```

```rust
async fn cmd_test(check_only: bool, keep: bool, ci: bool) -> Result<()> {
    let config = CorviaConfig::default().with_env_overrides();
    let is_lite = matches!(config.storage.store_type, corvia_common::config::StoreType::Lite);

    let introspect = Introspect::from_file_or_default(
        std::path::Path::new("tests/introspect.toml"),
    );

    // Phase 1: Check environment
    println!("  Checking environment...");
    if is_lite {
        println!("    LiteStore: no Docker required");
        println!("    Ollama: check manually (ollama serve)");
    } else {
        let docker = match DockerProvisioner::new() {
            Ok(d) => d,
            Err(e) => {
                eprintln!("  Infrastructure error: {e}");
                std::process::exit(2);
            }
        };
        let env_status = introspect.check_env(&docker).await;
        let mut all_running = true;
        for (service, running) in &env_status {
            if *running {
                println!("    {service}: running");
            } else {
                all_running = false;
                println!("    {service}: not running");
            }
        }
        if !all_running {
            println!("  Auto-provisioning missing services...");
            for (service, running) in &env_status {
                if !running {
                    let result = match *service {
                        "SurrealDB" => docker.start(
                            config.storage.surrealdb_user.as_deref().unwrap_or("root"),
                            config.storage.surrealdb_pass.as_deref().unwrap_or("root"),
                        ).await,
                        "vLLM" => docker.start_vllm(&config.embedding.model).await,
                        _ => Ok(()),
                    };
                    if let Err(e) = result {
                        eprintln!("  Infrastructure error: failed to provision {service}: {e}");
                        std::process::exit(2);
                    }
                    println!("    {service}: provisioned");
                }
            }
        }
    }

    if check_only {
        println!("  Environment check complete.");
        return Ok(());
    }

    // Phase 2: Create store for test
    let test_config = if is_lite {
        let mut tc = config.clone();
        let test_dir = std::env::temp_dir().join("corvia-introspect-test");
        tc.storage.data_dir = test_dir.to_string_lossy().to_string();
        tc
    } else {
        let mut tc = config.clone();
        tc.storage.surrealdb_ns = Some("corvia_introspect".into());
        tc.storage.surrealdb_db = Some("test".into());
        tc
    };

    let store = match corvia_kernel::create_store(&test_config).await {
        Ok(s) => s,
        Err(e) => {
            eprintln!("  Infrastructure error: {e}");
            std::process::exit(2);
        }
    };
    if let Err(e) = store.init_schema().await {
        eprintln!("  Infrastructure error: {e}");
        std::process::exit(2);
    }

    let engine = connect_engine(&config);
    let adapter = GitAdapter::new();

    println!("\n  Introspect: ingesting own source...");
    let chunks = introspect.ingest_self(".", &adapter, engine.as_ref(), store.as_ref()).await?;
    println!("    {chunks} chunks embedded and stored");

    // Phase 3: Self-query
    println!("\n  Introspect: running canonical queries...");
    let results = introspect.query_self(engine.as_ref(), store.as_ref()).await?;

    let report = corvia_kernel::introspect::IntrospectReport {
        results,
        chunks_ingested: chunks,
    };

    // Phase 4: Report
    println!();
    for r in &report.results {
        let status = if r.passed() { "pass" } else { "FAIL" };
        let actual = r.actual_file.as_deref().unwrap_or("(no results)");
        println!(
            "    [{status}] \"{}\"\n          expected: {}\n          actual:   {} (score: {:.3}, min: {:.3})",
            r.query_text, r.expect_file, actual, r.score, r.min_score
        );
    }

    println!(
        "\n  Introspect: {}/{} passed (avg score: {:.3})",
        report.pass_count(), report.results.len(), report.avg_score()
    );

    // Teardown
    if !keep {
        println!("  Cleaning up introspect data...");
        if let Err(e) = store.delete_scope(introspect.scope_id()).await {
            eprintln!("  Warning: teardown failed: {e}");
        } else {
            println!("  Teardown complete.");
        }
    }

    if ci && report.exit_code() != 0 {
        std::process::exit(report.exit_code());
    } else if !report.all_passed() {
        println!("\n  Some queries failed. Use --ci to fail with exit code 1.");
    }

    Ok(())
}
```

## Notes for the Implementer

1. **hnsw_rs API uncertainty**: The exact method signatures for `Hnsw::insert()`, `Hnsw::search()`, and `Neighbour` fields are based on docs research. If they don't match, consult [hnsw_rs docs](https://docs.rs/hnsw_rs/latest/hnsw_rs/) and adjust. The test-driven approach will catch mismatches immediately.

2. **Redb version**: Using redb 2.x. The `TableDefinition` API and transaction patterns are stable. If `range()` on `&str` keys doesn't work exactly as shown, check [redb docs](https://docs.rs/redb/latest/redb/).

3. **Ollama model name**: Ollama uses short model names (`nomic-embed-text`) not HuggingFace paths (`nomic-ai/nomic-embed-text-v1.5`). The config defaults reflect this difference.

4. **HNSW stale vectors**: When `delete_scope` is called, vectors remain in the HNSW index but their Redb entries are deleted. The post-filter in `search()` naturally excludes them. `corvia rebuild` cleans up stale vectors by reconstructing the index from scratch.

5. **SurrealDB compile time**: The `surrealdb` crate adds significant compile time. If working only on LiteStore code, use `cargo test -p corvia-kernel -- test_lite_store` to avoid recompiling the full workspace.
